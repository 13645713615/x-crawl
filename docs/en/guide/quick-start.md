# Quick Start

## Install x-crawl

::: tip Compatibility Note
x-crawl requires [Node.js](https://nodejs.org/en/) version 18+.
:::

::: code-group

```bash [NPM]
$ npm install x-crawl
```

```bash [Yarn]
$ yarn add x-crawl
```

```bash [PNPM]
$ pnpm add x-crawl
```

:::
